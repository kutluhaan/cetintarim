# Çetin Tarım Web Sitesi

Modern, tek sayfa React web sitesi - Bereketli Hasat, Çetin Çözümler

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm start
```

3. Aşağıdaki resim dosyalarını `public/` klasörüne kopyalayın:
   - `hazelnut (1).png` (Navigation logo için)
   - `ida-copper-PNG-768x512.png` (Hero section ürün resmi için)
   - `findik-hangi-illerimizde-yetistirilmektedir.jpg` (Hero section arka plan için)

## Teknolojiler

- React 18
- Tailwind CSS
- Lucide React (İkonlar)
- Google Fonts (Inter & Merriweather)
